### Exception

- 일반 예외(Exception) : 예외 처리 코드 여부를 검사하는 예외.
- 실행 예외(Runtime Exception) : 컴파일러가 예외 처리 코드 여부를 검사하지 않는 예외.

### 예외 처리 방법 두 가지

1. 직접 처리(try-catch)
2. 위임(throws)
